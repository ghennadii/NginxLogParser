# WMAdjustLogParser

## Compile

```
$ make
```
or
```
$ g++ main.cpp WMAdjustLogParser.cpp -std=c++14 -O2 -o parser
```

# Run
```
$ ./parser <input-file> <output-file>
```
```
$ ./parser <input-file>
```
or
```
$ ./parser
Please enter path of the log file: <input file>
Please enter path of the output file: <output file>
```
